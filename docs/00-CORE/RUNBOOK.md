# Runbook

## Workspace Commands (root)
- `npm run dev:web`
- `npm run dev:automation`
- `npm run build:web`
- `npm run build:automation`
- `npm run type-check`
- `npm run test:unit`
- `npm run test:wallet`
- `npm run test:pricing`
- `npm run test:shipments`

## Guardrails
- `npm run check:boundaries`
- `npm run check:file-size`
- `npm run ci:guardrails`

## Package Typecheck
- `npm run typecheck:core-types`
- `npm run typecheck:core-utils`
- `npm run typecheck:core-db`
- `npm run typecheck:domain-wallet`
- `npm run typecheck:domain-pricing`
- `npm run typecheck:domain-shipments`
- (e gli altri `typecheck:*` esposti nel root `package.json`)
