Aiutami a creare un nuovo file per questo progetto.

Prima chiedi:
1. Che tipo di file devo creare? (es. pagina, componente, funzione, configurazione)
2. Cosa deve fare questo file?
3. Dove va salvato nella struttura del progetto?

Poi crea il file con:
- Codice completo e funzionante
- Commenti in italiano per ogni parte importante
- Spiegazione finale di cosa ho davanti e come usarlo
