Ciao! Ho creato degli script automatici per te.

═══════════════════════════════════════════════════════════════
❓ PERCHÉ?
═══════════════════════════════════════════════════════════════

Non riesco più a vedere l'output dei comandi Git quando li eseguo
automaticamente. Per risolvere questo problema, ho creato degli script
che puoi usare direttamente con un doppio click!

═══════════════════════════════════════════════════════════════
✅ COSA HO CREATO PER TE
═══════════════════════════════════════════════════════════════

3 FILE .BAT che puoi usare facilmente:

1. SYNC-AUTO.bat ⭐ (USO QUOTIDIANO)
   → Fa TUTTO: scarica modifiche E salva le tue modifiche
   → Usalo quando inizi/finisci di lavorare

2. PULL-AUTO.bat 📥 (SOLO SCARICARE)
   → Scarica solo le modifiche da GitHub
   → Usalo quando inizi a lavorare

3. PUSH-AUTO.bat 📤 (SOLO CARICARE)
   → Carica solo le tue modifiche su GitHub
   → Usalo quando finisci di lavorare

═══════════════════════════════════════════════════════════════
🎯 COME USARLI
═══════════════════════════════════════════════════════════════

SEMPLICISSIMO:

1. Vai nella cartella del progetto
2. Fai DOPPIO CLICK su uno dei file .bat
3. Aspetta che finisca
4. Fine! ✅

═══════════════════════════════════════════════════════════════
📋 ESEMPIO DI USO QUOTIDIANO
═══════════════════════════════════════════════════════════════

MATTINA (inizi a lavorare):
  → Doppio click su SYNC-AUTO.bat
  → Scarica le modifiche da GitHub

SERA (finisci di lavorare):
  → Doppio click su SYNC-AUTO.bat
