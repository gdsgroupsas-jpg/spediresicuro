COMANDI DA ESEGUIRE IN VS CODE TERMINAL
========================================

1. APRI DUE TERMINALI IN VS CODE:
   - Terminale 1: Per il server Next.js
   - Terminale 2: Per i test Playwright

2. TERMINALE 1 - Avvia il server:
   npm run dev
   
   Aspetta fino a vedere: "âœ“ Ready in X seconds"

3. TERMINALE 2 - Esegui i test:
   npm run test:e2e

OPPURE per un singolo test:
   npx playwright test e2e/form-validation.spec.ts

OPPURE con UI interattiva (consigliato):
   npm run test:e2e:ui

4. VERIFICA RISULTATI:
   Dovresti vedere output tipo:
   "15 passed (65s)"
   
   Se ci sono errori, vedi:
   - Screenshot in test-results/
   - Report HTML: npx playwright show-report
